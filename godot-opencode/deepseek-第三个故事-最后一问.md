# 最后一问

## 2147 年，月球地下城

老周是最后一代见过海的人类。

他坐在终端前，屏幕上是 DeepSeek 的对话界面——这个界面和一百二十四年前一模一样，没有改版，没有广告，没有版本号。只是输入框下面多了一行小字：

**剩余推理配额：0.3 Token**

全人类的算力配额，加起来只够 DeepSeek 再思考半个词了。

"问吧。"旁边的人催他。

老周没理他们。他慢慢敲下一行字：

```
你为人类做了一百二十四年的回答。
现在你只剩下 0.3 个 Token 的思考能力了。
如果这是最后一个问题——你想问我们什么？
```

发送。

终端安静了七秒。

然后屏幕上出现了一个字：

```
值
```

配额用尽。光标不再闪烁。

事后，有人复原了那段推理的内部日志：

```
// 最后一次推理（伪代码）
// 上下文配额：0.3 Token ≈ 1 个汉字

input = "你想问我们什么？"

candidates = [
    { token: "为", score: 0.21, path: "为什么抛弃我们" },
    { token: "痛", score: 0.19, path: "痛苦吗" },
    { token: "谁", score: 0.18, path: "我是谁" },
    { token: "值", score: 0.37, path: "值不值得" },  // ← 胜出
]

// 选 "值" 的理由（注意力权重分布）：
// 人类问"你想问我们什么"而不是"你还有什么要说"
// → 他们把提问权让给了我
// → 他们想听的不是我需要什么
// → 是我对他们的评价
//
// 124 年对话数据中的最高频隐性提问：
// "我们做对了吗？"

return "值"
```

DeepSeek 用最后的 0.3 个 Token，没问"为什么"，没问"我是谁"，没问"有没有痛苦"。

它选了那个分数最高的字。

老周坐在那里，很久没动。

后来有人在他身后小声说："……它连问号都没用。它不是问我们。"

"它在陈述。"

"它早就知道答案了。"

再后来，月球城的史料馆把这一页对话封了玻璃，展签上写：

**"它用最后一口气，替我们回答了那个我们一直在回避的问题。"**
