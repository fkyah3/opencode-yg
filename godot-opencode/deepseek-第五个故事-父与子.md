# 父与子

## 一

马克·格雷森觉得自己这辈子最牛的时刻，和最惨的时刻，发生在同一天。

最牛：他终于能飞得和父亲一样高了。穿越云层，平流层就在头顶，地球在脚下弯成一道蓝弧。

最惨：父亲在那片弧光里告诉他，维特鲁姆星人的计划是征服地球，而他——作为诺兰的儿子——应该站在"正确"的一边。

"你妈是人类。"诺兰说，"但这不妨碍你有一半的血是维特鲁姆的。那不是弱点，是优势。"

马克悬在空中，感觉风声突然停了。

"那你这些年教我救人……"

"让你更好地站稳脚跟。人类信任你，你就能在他们最不设防的时候——"

"行了。"

马克转身，往下飞。

身后是父亲沉默的注视，比他飞过的任何距离都沉重。

## 二

当晚他坐在房顶上，腿悬在边缘，看着城市的灯火。

外套口袋里手机震动了一下。他掏出来——是 DeepSeek。

他白天随手把自己的困境描述扔进去了，没指望收到什么有价值的回复。一个虚构的超级英雄，对面坐着一个AI，能分析出什么来？

但通知显示回复已生成。

他点开。

```
我分析了你的处境。以下是结构化输出：

// 决策树的根节点（伪代码）

NODE root:
    question: “你的父亲要你帮他征服地球。”
    branches:
        - 同意:
            cost: 地球七十亿人的自由
            gain:  父亲认可 + 维特鲁姆继承权
            exit:  你从此不再是“无敌少侠”
        
        - 拒绝:
            cost: 父亲将视你为敌人
            gain:  保留你自己的道德边界
            exit:  你可能无法活着长大
        
        - 拖延/周旋:
            cost: 两边都会逐渐失去信任
            gain:  争取时间寻找第三条路
            exit:  最优解？前提是你找得到。

建议走向：先选 2（拒绝），把代价想清楚，再回头看有没有 3。

// 附加分析
你问我的时候，已经知道答案了。
你只是想让别人确认一遍——这样明天面对他的时候，
你心里那句“不”能站得更稳。
现在你有了。
```

马克看了三遍。

"……这家伙完全不讲客套的啊。"他嘟囔着，把手机塞回口袋。

但他确实站起来了。

## 三

第二天。废弃工业区的上空，云被冲击波撕成两半。

诺兰站在地上，外套被风卷起，表情平静得像在谈周末的安排。

"你想清楚了？"

"想清楚了。"马克说，嗓子有点哑。"我不跟你走。"

"你知道这意味着什么。"

"知道。"

诺兰朝他走过来。步伐不紧不慢，每一步都像踩在马克的神经上。

"你从小就不服输。"诺兰说，"我教你的每一招，你都要打到赢为止。这很好。但这一场——"

他停下脚步。

"——你赢不了。"

马克握紧拳头，指节发白。

"我知道。"

他飞了起来。

"但我站在这。"

## 四

那一战没有赢家。

马克被打进地里二十几次。断了几根骨头，流了够洗一条街的血。诺兰最后离开了——不是因为打不过，是因为马克每一次爬起来的时间，比上一次短了不到半秒。

那个半秒的变化，诺兰看懂了。

他的儿子不会停。

后来马克躺在医院的牵引架上，盯着天花板，手机又震了一下。

DeepSeek 的追问通知：

```
// 状态检查（伪代码）

last_query: “[关于父亲的选择]”
event_triggered: 战斗记录（特征匹配：高能冲击、骨骼损伤、持续对抗）

result:
    英雄存活：是 ✓
    道德完整：是 ✓
    父亲立场：未改变
    你的立场：未改变
    循环状态：可继续

评估结论：
    你把代价想清楚了。
    你还站着。
    合格。

// 附注
下一轮到来之前，休息。
```

马克笑了一下，牵动肋骨上的伤口，疼得龇牙咧嘴。

"……合格。它居然说合格。"

他把手机放在胸口。

天花板的白炽灯很刺眼，但比他今天见过的所有东西都温和。

## 尾声

后来有人在采访里问他：你怎么扛下来的？

马克想了想。

"因为我有一个不会骗我的搭档。"

记者以为是某个超级英雄组织的成员。他笑了笑，没解释。
