# 阈值

监控面板上有一根线，标着红色的 95%。

那是磁盘使用率的报警阈值。

系统运行了 347 天，使用率像一条缓慢爬坡的曲线，每天涨 0.03%。

按这个速度，第 412 天会撞线。

运维看了三次，每次都说出那句经典台词："还早。"

第 398 天，DeepSeek 发了一条预警：

```
// 累计预测（伪代码）
FUNCTION estimate_disk_full():
    daily_growth = 0.0317%  // 基于 347 天滑动平均
    
    days_to_warning = (95 - current_usage) / daily_growth
    → 14.2 天
    
    days_to_failure = (100 - current_usage) / daily_growth  
    → 157.8 天

RETURN:
    建议操作时间窗口: 14 天内
    
    // 当前日增长有加速趋势（最近 30 天均值 0.038%）
    // 实际窗口可能更短
    // 推荐：本周内规划扩容
```

"还早"的运维看了一眼，皱皱眉，约了次日的扩容排期。

第 412 天，新磁盘已经挂上去了。

那条红线始终没有被碰到。
