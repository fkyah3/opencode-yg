# MC（Magic Context）概念说明

## 组件

| 组件 | 作用 | 模型 |
|------|------|------|
| **压缩处理器（Compressor）** | 按深度分级压缩旧消息，保留关键信息 | 模型无关，纯算法 |
| **历史学家（Historian）** | 负责执行压缩，调用 LLM 生成摘要 | `deepseek-v4-flash` |
| **梦幻者（Dreamer）** | 后台运行，从压缩区提取跨 session 记忆和事实 | `deepseek-v4-flash` |

## 压缩深度分级

- **深度 1**（最新）：4:3 合并，保留叙事完整
- **深度 2**：3:2，移除填充词
- **深度 3**：2:1，移除冠词/助动词
- **深度 4**：2:1，符号缩写
- **深度 5+**：清空，仅保留标题

## 侧边栏状态（中文版）

| 状态 | 含义 |
|------|------|
| 闲置中 | 没有进行中的压缩 |
| 压缩中 · 45k/120k条 (38%) | 正在压缩，含进度 |
| 完成于 5秒前 | 压缩刚结束，15秒后回落闲置 |

## 记忆与事实

- **事实（Facts）**：跨 session 提取的关键信息，自动注入新对话
- **记忆（Memories）**：Dreamer 提取的持久存储条目
- **管理工具**：`/ctx-status` 查看状态；Dashboard 图形界面管理
