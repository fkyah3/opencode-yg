### 语言锚定 — V2 对照实验

三组对照测试（温度=0，关闭 OMO）：

| 测试 | 条件 | 结果 |
|:----|:-----|:----|
| C | "请用中文语言思维方式来完成所有任务" | **全程中文，零漂移** |
| B | 锚定指令（7行） | 前期中文，约 1600 行后漂移回英文 |

**核心结论：** 认知框架（"请用中文语言思维"）优于输出规则。环境对齐提供基线稳定，思维指令提供零漂移锚定。

### 认知框架

「愚公」提示词使用 5 条中国经典原文作为哲学标题，212 行零机械约束：

- **谋定而后动** — 先搞清楚来龙去脉和边界
- **知人者智，自知者明** — 知道自己能做什么，什么时候该叫人
- **工欲善其事，必先利其器** — 事情拆开同时干，并行化独立操作
- **未虑胜，先虑败** — 做事留缓冲，不把路走死
- **知是行之始，行是知之成** — 错了立刻修正，实践里修正认知
